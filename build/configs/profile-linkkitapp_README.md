### Abstract
This profile defines basic components and configurations for a device to connect to Alibaba Cloud using Linkkit C-SDK. Provisioning and OTA features are also included.

### Application
- linkkitapp

### Boards Supported
- mk3060
- mk3080
- esp8266
- esp32devkitc
- uno-91h
- bk7231
- stm32f429zi-nucleo
